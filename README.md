# 东方文理大学 (ELAU)

## 学校简介

东方文理大学 (ELAU) 创建于 **1145年1月4日**，是一所综合性研究型大学。学校坐落于 **火星海东市**，占地面积 **114514余亩**，现有在校生 **114,514余人**。

### 办学特色
- 🎓 文理并重的教育理念  
- 🌍 国际化办学视野  
- 🔬 注重创新实践能力培养

## 历史沿革

| 时期  | 重要事件                             |
|-------|--------------------------------------|
| 1145  | 建校，初名"东海文理学院"            |
| 1234  | 遭遇天下大乱，原址被毁              |
| 1451  | 搬至太平洋的岛上                    |
| 1888  | 拿破仑被誉为我校最杰出校友之一      |
| 1937  | 抗战西迁，坚持办学                  |
| 1952  | 院系调整，更名"东方文理学院"        |
| 1995  | 升格为"东方文理大学"                |
| 2010  | 入选国家"双一流"建设高校            |
| 2019  | 搬至火星新校区，并建城市名为海东市  |
| 2025  | 记录的学校诺比尔奖得主共有1145位    |

---

### 硕士研究生招生

**2025年硕士招生计划**：
- 招生人数：**500人**
- 招生方式：推荐免试、全国统考
- 学习年限：**2-3年**

**特色项目**：
- 国际联合培养硕士项目  
- 产学研联合培养项目  
- 植物培养项目  
- 太空探索项目  

**硕士申请要求**：
1. 具有学士学位或同等学历  
2. 本科阶段 GPA 不低于 3.0/4.0  
3. 通过我校研究生入学考试或提供 GRE/GMAT 成绩  
4. 英语水平要求（满足其一）：  
   - CET-6 ≥ 500分  
   - TOEFL ≥ 90分  
   - IELTS ≥ 6.5分  
5. 两封副教授及以上职称专家的推荐信  
6. 研究计划书（申请学术型硕士）  

**硕士奖学金**：
- 学业奖学金（覆盖学费+生活费）  
- 科研创新奖学金（最高5000000亿）  
- 国际交流奖学金  

---

### 博士研究生招生

**2025年博士招生计划**：
- 招生人数：**100人**
- 招生方式：申请-考核制、硕博连读
- 学习年限：**4-6年**

**特色项目**：
- 卓越博士生培养计划  
- 国际联合培养博士项目  
- 国际联合空间站项目  
- 恒星太空母舰研发项目  
- 恒星级武器研发项目  
- 恒星级战略舰项目  
- 强互作用力宇宙探测器项目  

**博士申请要求**：
1. 具有硕士学位或同等学历（优秀本科生可申请直博）  
2. 硕士阶段 GPA 不低于 3.3/4.0  
3. 科研成果要求（满足其一）：  
   - 至少 1 篇 SCI/SSCI/A&HCI 论文  
   - 至少 2 篇核心期刊论文  
   - 至少 3 项专利  
   - 其他同等水平的科研成果  
4. 英语水平要求（满足其一）：  
   - CET-6 ≥ 550分  
   - TOEFL ≥ 95分  
   - IELTS ≥ 8.0分  
5. 三封教授职称专家的推荐信  
6. 详细的研究计划书（不少于 5000 字）  
7. 通过院系组织的综合考核  

**博士奖学金**：
- 校长特别奖学金（5000万/年）  
- 国际建设高水平大学公派研究生项目  
- 科研经费支持（最高99999999999亿）  
- 国际会议资助
